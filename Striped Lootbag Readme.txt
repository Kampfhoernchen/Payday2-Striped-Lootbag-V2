How to install Striped Lootbag:

1. Unzip files
2. copy folder "Striped Lootbag" to:
   Steam\steamapps\common\PAYDAY2\assets\mod_overrides



How to uninstall Striped Lootbag:

Only delete folder "Striped Lootbag"



Striped Lootbag replaces:

- normal blue Lootbag
